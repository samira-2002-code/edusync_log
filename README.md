PHP Authentication System
 Description:
Ce projet est un système simple d'authentification en PHP.
Il permet l'inscription, la connexion et la gestion des sessions utilisateur.
 Structure du projet:
 =>
public/
- index.php
- login.php
- register.php
- dashboard.php

scripts/
- authprocess.php
- loginlogic.php
- logout.php
- database.php

 Fonctionnalités=>
- Inscription utilisateur
- Connexion utilisateur
- Gestion de session
- Protection du dashboard
- Déconnexion

 Utilisation=>
1. Lancer le serveur (XAMPP)
2. Ouvrir le projet dans le navigateur
3. S'inscrire puis se connecter
4. Accéder au dashboard
